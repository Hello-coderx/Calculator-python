

*Tkinter Calculator*  
A desktop calculator application built using Python and the Tkinter library.

*Project Description*  
This program provides a graphical user interface for performing standard mathematical calculations. It features a responsive layout with a dark theme and support for various operations including square roots and percentages.

*Features*
- Basic Arithmetic: Addition, Subtraction, Multiplication, Division
- Advanced Operations: Square Root, Percentage, Sign Change (+/-)
- Clear Function: AC button to reset the calculator
- User Interface: Custom styled buttons and display using Tkinter grid geometry

*Requirements*
- Python 3.x
- Tkinter (included in standard Python installation)

*Installation and Usage*
1. Copy the source code into a file named http://calculator.py.
2. Open your terminal or command prompt.
3. Run the application using the following command: 
python http://calculator.py

*Technical Details*  
The application is built using a class-based approach (Object-Oriented Programming). It utilizes string manipulation and the math library for expression processing, and http://tk.StringVar for dynamic display updates.
